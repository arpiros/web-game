# Game Improvement Plan

> 기준 버전: Dark Fantasy Roguelike  
> 이 문서는 완료된 작업과 남은 개발 항목을 한 곳에서 관리한다.

---

## 현재 구현 상태

| 영역 | 구현 완료 |
|------|-----------|
| 캐릭터 | 5종 (칠흑의 기사, 불꽃의 마법사, 빛의 성기사, 조류 무희, 광전사) |
| 스킬 | 42종 (물리/화염/수/어둠/빛/유틸 + 캐릭터 전용 10종) |
| 적 | 16종 (early 3+1 / mid 4+2 / late 3+2 / 보스 2) + 엘리트 3종 |
| 동료 | 12종 (기본 8 + 신규 4) |
| 아이템 | 27종 (기본 15 + 신규 12) |
| 상태이상 | 12종 (poison, burn, freeze, stun, shield, regen, powerup, defdown, mana_regen, cc_immune, undying, miss_immune) |
| 라운드 | 7라운드 (라운드 3·5 엘리트, 라운드 7 보스 dragon_lord) |
| 드래프트 | 전투 후 3장 선택 (스킬/동료/아이템) + 아이템 조합(Forge) 탭 |
| MP 시스템 | 자동재생(+3/턴) + 턴 종료보너스(+8) + 킬보너스(+5) + mana_regen + mp_regen 아이템 |

## ✅ 완료된 항목

### 핵심 전투 시스템
- [x] **자동 MP 재생 + 턴 종료 MP 보너스** — 패시브 +3/턴, 턴 종료 클릭 시 +8
- [x] **크리티컬 / 미스 시스템** — 크리티컬 확률 10~25% (속도 기반), 미스 5~15% (방어력 기반)
- [x] **데미지 팝업 (Floating Numbers)** — 피해/크리티컬/미스/회복 색상 구분, 위로 이동 + 페이드아웃
- [x] **0MP 기본 공격** — 각 캐릭터별 원소 기본기 (MP 고갈 시 항상 사용 가능)
- [x] **힐 단일 회복 상한** — 단일 힐이 최대 HP의 40% 초과 불가

### 콘텐츠 확장
- [x] **신규 스킬 +10종** — `void_blade`, `blood_pact`, `mana_burst`, `phoenix_rebirth`, `holy_aura`, `divine_judgment`, `tsunami_dance`, `current_step`, `bloodlust`, `death_charge`
- [x] **신규 아이템 +12종** — `vampire_ring`, `ancient_scroll`, `elemental_core`, `storm_cloak`, `cursed_necklace`, `revival_potion`, `magic_antidote`, `time_sand`, `heroes_crest`, `twin_wings` 등
- [x] **신규 적 +6종** — `skeleton_archer`, `flame_phoenix`, `dark_vampire`, `frost_giant`, `poison_hydra`, `void_lord`
- [x] **신규 동료 +4종** — `storm_mage`, `guardian_angel`, `poison_witch`, `battle_bard`

### 전투 피드백 & 조작감
- [x] **전투 속도 조절** — HUD에서 1x / 1.5x / 2x 토글, 애니메이션 딜레이 배율 적용
- [x] **배틀 로그 최신순 정렬** — 최신 항목이 위쪽, 스킬 바를 HUD 바로 아래로 이동
- [x] **전투 화면 커스텀 툴팁** — 스킬 버튼 hover 시 속성·설명·MP/쿨다운·예상 데미지 표시
- [x] **상태이상 배지 툴팁** — hover/tap 시 상태이상 이름·지속 턴·효과값 표시
- [x] **피격 플래시 애니메이션** — 피해 시 빨간 glow 플래시 (`entity-hit-flash`)
- [x] **크리티컬 팝업 강화** — 전용 애니메이션(`popup-float-crit`) + ★ 접두사

### 게임플레이 구조
- [x] **원소 시너지 시스템** — 4종 시너지: 증기 폭발/카오스/파멸의 일격/성수. 드래프트 카드에 "✦ {시너지명}" 배지
- [x] **캐릭터별 가중치 드래프트** — `CharacterDef.draftWeights`로 테마 스킬/아이템/동료 가중치 설정
- [x] **이벤트 시스템** — 라운드 2·4 승리 후 이벤트 4종 (`abandoned_campfire`, `cursed_altar`, `wandering_merchant`, `ancient_library`)
- [x] **엘리트 전투 시스템** — 라운드 3·5에 엘리트 조우. 스탯 ×1.5, 처치 시 드래프트 4장
- [x] **보스 다단계 패턴** — `dragon_lord` HP 비율별 3페이즈 행동 전환
- [x] **동료 타겟팅** — 적이 동료를 공격 타겟으로 포함 (`attack`, `attack_all`)
- [x] **전투 클리어 시 HP 회복** — 드래프트 진입 시 최대 HP의 25% 회복

### 드래프트 & UX
- [x] **스킬/아이템 조합(Forge) 시스템** — 드래프트 화면에 "아이템 조합" 탭, 재료 2개 → 결과물 조합. 레시피 16종
- [x] **드래프트 소지 여부 배지** — 이미 보유한 스킬·아이템 카드에 "보유중" 배지
- [x] **동료 카드 행동 정보** — AllyCard에 공격 배율·속성·행동 설명 표시

### 통계 & 기록
- [x] **런 스코어 & 통계** — 결과 화면에 누적 피해·라운드·클리어 여부 표시

## 🔲 미완료 항목

### Phase 3 — 게임플레이 구조 확장

#### 3-4. HP 회복 선택지 추가
드래프트를 포기하고 HP를 회복하는 선택지를 추가한다.

```
옵션 A: [드래프트 대신 회복]  HP 최대치의 30% 회복 선택 가능
옵션 B: [안식의 불꽃]         이벤트 중 하나로 회복 샘 추가 (이벤트 시스템 연계)
옵션 C: [아이템 효과]         "전투 후 HP 소량 회복" 아이템 추가
```

---

### Phase 4 — 메타 진행 시스템

#### 4-2. 업적 시스템
```
업적 예시:
  [불사신]       HP 100 이하로 생존하며 전투 클리어
  [완벽한 일격]  크리티컬로 보스에게 1000 이상 피해
  [영웅의 귀환]  5번의 런 완주
  [마나 마스터]  MP를 고갈시키지 않고 클리어
  [원소 지배자]  모든 원소 속성 스킬 사용
  [무결점]       미스 없이 클리어
```
**신규 파일**: `src/state/achievementStore.ts`, `src/screens/AchievementScreen.tsx`

#### 4-3. 도전 모드
```
조건 예시:
  [이단자 모드]  아이템 드래프트 없음
  [광기 모드]    적 HP 2x, 적 공격력 1.5x
  [제한 모드]    스킬 슬롯 최대 3개
  [순수 모드]    동료 없이 단독 클리어
```

#### 4-4. 캐릭터 잠금 해제
```
해금 조건 예시:
  빛의 성기사:  첫 번째 클리어
  조류 무희:    물 원소 적을 50마리 처치
  광전사:       HP 200 이하로 보스 처치
```
**변경 파일**: `src/state/runStore.ts`, `src/screens/CharacterSelectScreen.tsx`

---

### Phase 5 — UI/UX 완성도

#### 5-1. 설정 메뉴
```
항목:
  - 전투 속도 (1x / 1.5x / 2x)
  - 배틀 로그 크기 (10 / 20 / 30줄)
  - 애니메이션 ON/OFF
  - 데미지 팝업 ON/OFF
```
**신규 파일**: `src/screens/SettingsScreen.tsx`

#### 5-2. 인벤토리 / 파티 정보 패널
```
추가할 정보:
  - 보유 아이템 전체 목록 (드래프트 화면 내)
  - 현재 파티 스탯 합산 보기
  - 현재 적용 중인 아이템 효과 목록
```

#### 5-3. 튜토리얼 / 첫 플레이 가이드
```
구성:
  - 캐릭터 선택 화면: 각 캐릭터 플레이스타일 설명
  - 첫 번째 전투: 단계별 힌트 오버레이
  - 드래프트 화면: 카드 유형 설명
  - 원소 상성 간략 안내 차트
```

#### 5-4. 사운드 효과
```
효과음 대상:
  - 스킬 사용 (원소별 다른 효과음)
  - 크리티컬 히트 / 미스
  - 적 처치
  - 드래프트 카드 선택
  - 보스 등장 / 승리 / 패배
```
Web Audio API 또는 Howler.js 사용.

#### 5-5. 키보드 단축키
```
전투: 1~6 스킬 선택, Enter 첫 번째 적 공격, Tab 다음 타겟, Space 턴 종료
드래프트: 1~3 카드 선택
```

---

### Phase 6 — 고급 시스템 (아이디어 단계)

#### 6-1. 번개 원소 추가
현재 5원소(physical, fire, water, dark, light)에 번개 추가.
강함: water(전도), physical / 고유 효과: 연쇄 공격 (적중 시 30% 확률로 인접 적 50% 피해)

#### 6-2. 스킬 강화 시스템
드래프트에서 이미 보유한 스킬 재선택 시 강화.
```
레벨 1 → 2: damage +0.3배율 / heal +0.5배율 / apply_status +1턴
```

#### 6-3. 동료 관계 시스템
```
조합 예시:
  [성직자 + 성기사]   치유량 +20%
  [암살자 + 궁수]     크리티컬 확률 공유 (+5%)
  [화염 정령 + 고대 용] 화염 피해 +15%
```

#### 6-4. 무한 모드
7라운드 클리어 후 계속 진행. 라운드 8 이상: 적 스탯 +15%/라운드, 특수 보스 사이클.
