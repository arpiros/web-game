# 개선사항 & 남은 작업

## 완료

- [x] **캐릭터별 가중치 드래프트** (Phase 1-5): `CharacterDef.draftWeights`로 캐릭터 테마 스킬/아이템/동료가 더 자주 등장. 가중치 5(테마 핵심) ~ 3(상황별 유용), 미지정은 원소 친화도(x3) 또는 기본값 1

- [x] **동료 타겟팅**: 적이 동료를 공격 타겟으로 삼도록 구현 (`attack`, `attack_all` 모두 파티원+동료 통합 타겟 풀 사용)
- [x] **전투 클리어 시 HP 회복**: `completeBattle`에서 드래프트 진입 시 최대 HP의 25% 회복 (치유 없는 캐릭터 지속 가능성 개선)
- [x] **힐 단일 회복 상한**: `combat.ts` heal 케이스에서 단일 힐이 최대 HP의 40%를 초과하지 않도록 제한 (무한버티기 방지)
- [x] **명상 스킬 쿨다운 조정**: 쿨다운 2턴 → 3턴 (마나회복 순환 속도 완화)
- [x] **런 스코어 & 통계** (Phase 4-1): 전투 결과 화면에 누적 피해·라운드·클리어 여부 표시
- [x] **보스 다단계 패턴** (Phase 3-3): `bossPhases` 기반 HP 비율별 행동 패턴 전환 (phase 1→2→3)
- [x] **전투 속도 조절** (Phase 1-6): HUD에 1x / 1.5x / 2x 토글 버튼, 애니메이션 지연 배율 적용
- [x] **스킬/아이템 조합(Forge) 시스템**: 드래프트 화면에 "아이템 조합" 탭 추가. 재료 2개 → 전설급 결과물 조합 가능. 레시피 8종 구현
- [x] **드래프트 소지 여부 배지**: 이미 보유한 스킬·아이템 카드에 "보유중" 배지 표시
- [x] **전투 화면 커스텀 툴팁**: 스킬 버튼 hover 시 속성·설명·MP/쿨다운·예상 데미지 표시
- [x] **배틀 로그 최신순 정렬**: 로그 최신 항목이 위쪽에 오도록, 스킬 바를 HUD 바로 아래로 이동
- [x] **동료 카드 행동 정보**: 드래프트의 AllyCard에 공격 배율·속성·행동 설명 표시
- [x] **상태이상 배지 툴팁**: hover/tap 시 상태이상 이름·지속 턴·효과값 표시
- [x] **원소 시너지 시스템** (Phase 1-4): 4종 시너지 전투 효과 (`synergy.ts`), 드래프트 카드에 "✦ {시너지명}" 배지 표시 (선택 시 새로 활성화될 시너지 미리 알림)
- [x] **엘리트 전투 시스템** (Phase 3-2): 라운드 3·5에 엘리트 조우 추가. `EnemyDef.tier: 'elite'`, 엘리트 3종 (`iron_golem`, `shadow_stalker`, `volcanic_wyvern`). 스탯 ×1.5 (라운드 스케일 위), 처치 시 드래프트 4장 제시, EnemyCard에 "⚔ 엘리트" 배지 표시
- [x] **이벤트 시스템** (Phase 3-1): 라운드 2·4 전투 승리 후 드래프트 전에 이벤트 화면 등장. 이벤트 4종 (`abandoned_campfire`, `cursed_altar`, `wandering_merchant`, `ancient_library`). 각 이벤트마다 선택지 2개, 효과: HP 회복·스탯 변경·스킬 획득·아이템 획득. `RunPhase`에 `'event'` 추가, `EventScreen` 신규 컴포넌트
